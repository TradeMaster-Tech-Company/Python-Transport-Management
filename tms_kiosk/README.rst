======================================
Transport Management System Kiosk Mode
======================================

This application allows you to scan truck with barcode and validate if it's able to leave the company.

Configuration
=============

To configure this module, you need to:

#. Go to vehícle catalog.
#. Add the content to scan in the menu ...

Usage
=====

To use this module, you need to:

#. Open Kiosk mode.
#. Read the vehícle, it will show the documentation needet to validate.
